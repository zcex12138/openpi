# Franka 主链使用说明

## 1. 启动推理服务

```bash
source /home/mpi/workspace/yhx/openpi/.venv/bin/activate
uv run scripts/serve_policy.py \
    policy:checkpoint \
    --policy.config=pi05_franka_cola_relative_lora \
    --policy.dir=data/checkpoints/pi05_franka_cola_relative_lora/20260305_153045/4000
```

## 2. 启动相机服务

```bash
PYTHONPATH=/home/mpi/workspace/yhx/openpi \
python examples/franka/camera_service.py
```

仅排障时再使用相机可视化脚本：

```bash
PYTHONPATH=/home/mpi/workspace/yhx/openpi \
python examples/franka/camera_test_visualize.py
```

## 3. 运行评估 / 录制

```bash
PYTHONPATH=/home/mpi/workspace/yhx/openpi \
python examples/franka/main.py \
    --args.remote-host localhost --args.remote-port 8000 \
    --args.open-loop-horizon 30 \
    --args.control-fps 30 \
    --args.control-mode impedance
```

临时覆盖录制目录：

```bash
PYTHONPATH=/home/mpi/workspace/yhx/openpi \
python examples/franka/main.py \
    --args.remote-host localhost --args.remote-port 8000 \
    --args.max-episode-time 120 \
    --args.open-loop-horizon 10 \
    --args.control-fps 30 \
    --args.record-pkl \
    --args.record-dir eval_records/pi05_franka_position_control_lora \
    --args.control-mode impedance
```

如果使用本地 checkpoint，保留原有 CLI 覆盖方式：

```bash
PYTHONPATH=/home/mpi/workspace/yhx/openpi \
python examples/franka/main.py \
    --args.checkpoint-dir data/checkpoints/pi05_franka_position_control_lora/<run>/<step> \
    --args.config pi05_franka_position_control_lora
```

## 4. PKL 转 Zarr

默认参数现在直接写在 `examples/franka/convert_pkl_to_zarr.py` 顶部的 `SCRIPT_DEFAULT_ARGS` 里。先直接修改这些值，再运行：

```bash
/home/mpi/workspace/yhx/openpi/examples/franka/.venv/bin/python \
    /home/mpi/workspace/yhx/openpi/examples/franka/convert_pkl_to_zarr.py
```

如果只想临时覆盖某个参数，仍然可以直接加命令行参数；例如指定输入目录并改用 `corrected` 作为导出的 `data/action`：

```bash
/home/mpi/workspace/yhx/openpi/examples/franka/.venv/bin/python \
    /home/mpi/workspace/yhx/openpi/examples/franka/convert_pkl_to_zarr.py \
    --input-dir eval_records/pi05_franka_position_control_lora \
    --action-target corrected
```
